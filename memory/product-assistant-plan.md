## 二、分阶段实施计划

### 第一阶段：竞品情报雷达（Week 1-2）
**目标：** 每天自动生成竞品情报简报

**已有基础：** `background-intelligence-report` + `aerial-work-platform-analysis`
**需要建设：**
- [ ] 竞品列表配置化（临工重机/海德吉斯/星邦智能/欧历胜等）
- [ ] 定时任务（每日早上 8 点自动运行）
- [ ] 输出格式优化（简报样式，推送至企业微信）

**交付物：** `competitor-radar.py` + cron 任务

---

### 第二阶段：行业信号降噪系统（Week 2-3）
**目标：** 从噪音中提取高价值信号

**技术方案：**
- 关键词订阅机制（高空作业平台/塔机/工程机械等）
- 多源采集：政策（住建部/应急部）、媒体（工程机械周刊/建筑时报）、投资（天眼查/烯牛数据）
- 过滤规则：去重 / 相关性打分 / 信号分级（🔴重要/🟡关注/⚪参考）
- 输出：每日「行业信号日报」

**交付物：** `industry-signal-filter.py` + 推送机制

---

### 第三阶段：深度研报白话翻译（Week 3-4）
**目标：** 一键解析研报/论文核心观点

**技术方案：**
- 输入：PDF 或 URL
- 处理：RAGFlow 解析 → 结构化提取 → 大模型生成摘要
- 输出：核心结论 + 关键数据 + 一句话评价 + 阅读建议

**交付物：** `research-paper-summarizer.py` 或封装为 skill

---

### 第四阶段：结构化文档生成（Week 4-5）
**目标：** 说想法，自动生成规范文档

**技术方案：**
- 模板引擎：PRD / 周报 / 月报 / 会议纪要 / 产品介绍
- 工作流：输入意图 → 提取关键信息 → 填充模板 → 输出
- 企业格式统一（公司标题/编号/审批流程）

**交付物：** `doc-generator.py` skill 或模板集合

---

### 第五阶段：文本驱动原型速绘（Week 5-6）
**目标：** 描述页面需求 → 输出 HTML/Figma

**技术方案：**
- 自然语言理解 → 结构化组件描述
- 输出 HTML 原型（可交互）或 Figma -compatible JSON
- 适合移动端 H5 页面快速验证

**交付物：** `prototype-generator.py`

---

### 第六阶段：PPT 一键生成（Week 6-7）
**目标：** 数据 + 意图 → 完整汇报 PPT

**技术方案：**
- 输入：Excel 数据 / Excel 文件 + 汇报主题
- 自动生成：封面 + 目录 + 数据页 + 分析页 + 结论 + 行动建议
- 支持企业微信推送或本地下载

**交付物：** `ppt-generator.py`（基于 python-pptx）
