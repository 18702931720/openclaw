# 🏗️ 高空作业平台行业政策分析技能

## 一、技能定位

专为高空作业平台（Aerial Work Platform，简称AWP）制造商及相关企业设计的政策分析工具，深度服务星邦智能等行业玩家。

**适用场景：**
- 行业政策检索与解读
- 竞争对手商务动作分析
- 合规体系建设
- 制度文件编写
- 业务流程分析（人机物法环）
- 事前-事中-事后全生命周期管理
- 政策影响评估与穿透分析

**触发词：** 高空作业平台政策、行业政策分析、政策检索、竞争情报、竞对分析、行业动态、政策报告生成、制度文件编写、业务流程分析、合规分析

## 二、核心工作流程（三阶段）

### Phase 1：政策与情报检索采集

**检索渠道优先级：**
1. 政府官网（住建部/应急管理部/市场监管总局/工信部）→ 网页快照保存
2. 国家标准信息公共服务平台 → GB标准检索
3. 行业媒体（工程机械杂志、建筑时报、安全生产报）
4. 竞争对手官网/公告（定期监控）
5. 行业协会/联盟网站

**本地存储结构：**
```
桌面/高空作业平台政策库/
├── 2026年度/
│   ├── 安全生产类/
│   ├── 产品标准类/
│   ├── 行业准入类/
│   ├── 环保政策类/
│   └── 竞争对手情报/
└── 政策索引表.xlsx
```

**索引表字段：**
政策名称、文号、发布部门、发布日期、施行日期、政策类型、相关上位法、主要内容摘要、对本行业的影响程度（高/中/低）、本地文件路径、网页快照链接、检索日期

### Phase 2：政策与情报分析（核心方法论）

#### 2.1 人机物法环分析矩阵

| 维度 | 分析内容 | 高空作业平台场景示例 |
|------|----------|---------------------|
| 人（人员） | 责任部门、人员资质、培训要求 | 操作人员证、维保人员资质要求 |
| 机（设备） | 设备配置、系统建设、产品标准 | 防坠落装置、限高限重系统、产品认证 |
| 物（物料） | 原材料、供应链、零部件 | 液压元器件、钢材、电池系统 |
| 法（法规） | 上位法依据、配套制度、国家标准 | 特种设备安全法、GB标准、行业规范 |
| 环（环境） | 使用场景、市场环境、监管环境 | 施工现场规范、租赁行业政策、海外市场法规 |

#### 2.2 事前-事中-事后管理矩阵

| 阶段 | 管理要点 | 责任部门 | 完成时限 |
|------|----------|----------|----------|
| 事前 | 政策跟踪、标准预研、合规预判 | 战略发展部/法务部 | 政策发布前6个月 |
| 事中 | 政策解读影响、应对方案制定 | 战略发展部/市场部 | 政策发布后1个月内 |
| 事后 | 制度更新、产品调整、市场策略 | 各业务部门 | 按项目周期 |

#### 2.3 竞争对手分析维度（行业专用）

| 分析维度 | 关注内容 | 信息来源 |
|----------|----------|----------|
| 产品动态 | 新品发布、技术迭代、价格调整 | 官网、行业媒体、展会 |
| 商务动作 | 渠道拓展、战略合作、并购 | 公告、行业新闻 |
| 政策响应 | 产品合规调整、市场策略变化 | 官网公告、投资者问答 |
| 市场份额 | 销售额、出货量、市场排名 | 行业协会数据、上市公司财报 |
| 舆情监控 | 负面投诉、安全事故、质量问题 | 社交媒体、消费者投诉平台 |

#### 2.4 指标体系设计规范

**核心原则：** 指标设计必须以政策原文和行业数据为依据，严格区分强制性指标与参考性指标。

**"必须"类指标处理规则：**

| 场景 | 达标率默认值 | 说明 |
|------|-------------|------|
| 政策明确要求"必须"做到的事项 | 100% | 无条件达标，不设容错空间 |
| 政策有明确范围或比例要求 | 按政策明确范围 | 如"不超过XX%"、"不低于XX%" |
| 政策无具体数值但属于"必须"项 | 100% | 视为零容忍项 |
| 政策明确引用参考指南的范围 | 按参考指南范围 | 如国家标准GB规范 |

**达标率设计示例：**

【强制类指标 - 100%达标】
- ✓ 产品认证覆盖率 = 100%（必须通过认证才能销售）
- ✓ 安全装置配置率 = 100%（防坠落、急停等）
- ✓ 标准合规检查完成率 = 100%

【范围类指标 - 按政策范围设计】
- ✓ 产品召回响应时间 ≤ 24小时
- ✓ 客户投诉处理及时率 ≥ 95%
- ✓ 售后服务满意度 ≥ 90%

【行业对标类指标 - 按行业基准设计】
- ✓ 市场占有率目标 ≥ 行业前三
- ✓ 新品研发周期 ≤ 12个月
- ✓ 展会参与覆盖率 ≥ 80%

**指标表格字段：**
| 指标名称 | 指标定义 | 计算公式 | 达标阈值 | 达标率 | 指标来源 | 性质判定 |

## 三、分析场景说明

### 场景1：政策影响分析
当有重大政策发布时（如GB标准修订、安全监管加强）：
1. 检索政策原文和解读
2. 评估对产品研发、生产、销售的影响
3. 制定应对方案和时间表
4. 输出政策影响评估报告

### 场景2：竞争对手监控
定期监控主要竞争对手（如临工重机、星邦智能、徐工、中联等）：
1. 监控新产品发布、价格调整、渠道变化
2. 分析市场策略变化和背后逻辑
3. 评估对本公司市场地位的影响
4. 输出竞争情报周报/月报

### 场景3：合规体系建设
针对目标市场（如国内、海外）建立合规体系：
1. 梳理目标市场的法规和标准要求
2. 对照现有产品进行差距分析
3. 制定整改和认证计划
4. 输出合规整改路线图

## 五、注意事项

1. **以"我司"视角：** 当分析星邦智能时，以"我司"自称进行行业对比
2. **竞对情报合规：** 所有情报收集限于公开信息来源，不涉及商业秘密
3. **政策时效性：** 定期更新政策库，及时清理过期政策信息
4. **交叉验证：** 竞对动态需多源验证，确保信息准确性
5. **保密原则：** 分析报告仅供内部使用，注意信息安全
