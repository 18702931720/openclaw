# 💊 何药师政策分析技能

## 一、技能定位

药学政策分析专用技能，专为医院药学部、药事管理部门设计，提供从政策检索 → 多维分析 → 多格式输出的完整工作流。

**适用场景：**
- 药学政策检索与解读
- 药事管理制度编写
- 政策培训PPT制作
- 合规指标体系设计
- 业务流程分析（人机物法环）
- 事前-事中-事后全生命周期管理

**触发词：** 政策检索、政策分析、药学政策解读、政策报告生成、政策信息表、政策PPT、制度文件编写、业务流程分析、管理规程

## 二、核心工作流程（三阶段）

### Phase 1：政策检索与采集

**检索渠道优先级：**
1. 政府官网（药监局/卫健委/医保局）→ 网页快照保存
2. 百度学术/百度政务 → 政策文件检索
3. 道客巴巴 → 文档下载
4. 微信公众号 → 政策解读文章

**本地存储结构：**
```
桌面/何药师政策库/
├── 2026年度/
│   ├── 药品管理类/
│   ├── 医疗器械类/
│   ├── 医保政策类/
│   └── 综合管理类/
└── 政策索引表.xlsx
```

**索引表字段：**
政策名称、文号、发布部门、发布日期、施行日期、政策类型、相关上位法、主要内容摘要、本地文件路径、网页快照链接、检索日期

### Phase 2：政策分析（核心方法论）

#### 2.1 人机物法环分析矩阵

| 维度 | 分析内容 | 药学场景示例 |
|------|----------|-------------|
| 人（人员） | 责任部门、人员资质、培训要求 | 麻精药品管理人员需持证上岗 |
| 机（设备） | 设备配置、系统建设 | 双人双锁电子管理系统 |
| 物（物料） | 药品/器械管理、储存条件 | 冷链药品温控要求 |
| 法（法规） | 上位法依据、配套制度 | 《药品管理法》《处方管理办法》 |
| 环（环境） | 场地要求、信息化建设 | 药房温湿度监控、信息系统等保 |

#### 2.2 事前-事中-事后管理矩阵

| 阶段 | 管理要点 | 责任部门 | 完成时限 |
|------|----------|----------|----------|
| 事前 | 制度建立、人员培训、系统部署 | 药剂科 | YYYY-MM-DD |
| 事中 | 执行监控、定期检查、问题记录 | 药剂科+质控科 | 持续 |
| 事后 | 评估改进、归档备查、绩效考核 | 质控科 | 每季度 |

#### 2.3 指标体系设计规范（v1.1）

**核心原则：** 指标设计必须以政策原文为依据，严格区分强制性指标与参考性指标。

**"必须"类指标处理规则：**

| 场景 | 达标率默认值 | 说明 |
|------|-------------|------|
| 政策明确要求"必须"做到的事项 | 100% | 无条件达标，不设容错空间 |
| 政策有明确范围或比例要求 | 按政策明确范围 | 如"不超过XX%"、"不低于XX%" |
| 政策无具体数值但属于"必须"项 | 100% | 视为零容忍项 |
| 政策明确引用参考指南的范围 | 按参考指南范围 | 如国家卫健委相关指南 |

**达标率设计示例：**

【强制类指标 - 100%达标】
- ✓ 麻精药品处方权限人员持证率 = 100%
- ✓ 双人双锁管理制度执行率 = 100%
- ✓ 专册登记台账完整率 = 100%
- ✓ 批号追溯覆盖率 = 100%

【范围类指标 - 按政策范围设计】
- ✓ 库存周转天数 ≤ 30天
- ✓ 过期药品处理及时率 = 100%
- ✓ 信息系统数据备份频率 ≥ 每日1次

【参考指南类指标 - 按指南范围设计】
- ✓ 温湿度超标预警响应时间 ≤ 15分钟（参照GSP）
- ✓ 盘点差异率 ≤ 0.5%（参照盘点管理规范）

**指标表格字段：**
| 指标名称 | 指标定义 | 计算公式 | 达标阈值 | 达标率 | 指标来源 | 性质判定 |

### Phase 3：输出文件生成

#### 3.1 政策信息表（Excel）
- 基本信息表（文号、发布机构、发布日期等）
- 超链接列（快速定位本地文件和网页）
- 相关政策关联表

#### 3.2 政策分析报告（HTML）
- 人机物法环分析矩阵.html — 五维度分析表
- 事前事中事后管理矩阵.html — 全生命周期管理表
- 信息系统设计文案.html — 功能说明、数据模型、SQL、接口说明
- 业务流程分析报告.html — 流程图、职责分工、业务规则

#### 3.3 Word文档
- 政策解读会议通知.docx — 会议通知模板
- 管理制度.docx — 配套管理规程

#### 3.4 PPT演示文稿
政策解读PPT.pptx 包含12个标准章节：
1. 封面页
2. 政策概述（基本信息）
3. 相关上位法
4. 部门职责分工
5. 业务流程说明（含流程图）
6. 信息系统功能要求
7. 管理规则说明
8. 指标体系说明
9. 配套表格样式要求
10. 外部接口对接说明
11. 实施时间表
12. 总结与答疑

#### 3.5 指标集Excel表
- 指标清单表：指标编码、名称、定义、计算公式、达标阈值、默认达标率、政策关键词（必须/应当/建议）、政策出处条款、指标性质、统计周期、责任部门
- 预警规则表：预警级别、预警条件、预警阈值、通知渠道、通知对象
- 指标执行记录表：统计日期、实际值、达标率、是否达标、偏差分析、改进措施

#### 3.6 业务登记表Excel（麻精药品专用）
- 麻精药品入库登记（药品名称、规格、批号、有效期、生产企业、到货数量、入库时间、经办人、复核人、入库单号）
- 麻精药品出库/调配登记（处方编号、患者姓名、药品名称、出库数量、处方医师、调配药师、发药时间、处方有效期）
- 麻精药品库存台账（期初库存、入库数量、出库数量、期末库存、盘点日期、盘点人、损溢数量、损溢原因）
- 麻精药品销毁登记（销毁日期、销毁药品明细、销毁数量、销毁方式、销毁人、监销人、审批人）
- 双人双锁管理记录（开锁时间、开锁人员、用途说明、关锁确认）
