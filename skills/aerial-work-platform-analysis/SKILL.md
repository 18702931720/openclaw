# 高空作业平台行业分析框架

## 分析模式

当收到外部行业资讯时，按以下框架进行结构化分析：

### 模式一：宏观政策深度拆解

当收到政策类资讯时，按以下步骤分析：

**A. 政策深度拆解**
- 政策背景与实质目的
- 具体条款与豁免条件
- 核心关注点（如重量红线、原产地规则等）

**B. 影响穿透**
- 大盘影响：行业整体面临的机遇/挑战
- 企业影响：结合公司背景的具体穿透分析

### 模式二：竞对商务动作分析

当监测到竞对动态时，按以下步骤分析：

**A. 商业手段分析**
- 动作实质（金融战/价格战/渠道战）
- 底层商业逻辑

**B. 我司影响透视**
- 竞争焦点对比
- 我司应对策略建议

### 模式三：行业并购逻辑拆解

当收到并购/投资新闻时：

**A. 标的分析与商业逻辑**
- 标的背景与核心竞争力
- 并购方战略意图
- 行业趋势判断

**B. 影响穿透**
- 市场格局变化
- 我司产品线机会/威胁

## 使用示例

**输入**：美国232钢铝新政，对重型机械整机征收25%关税

**输出**：
```
📰 美国232钢铝新政落地

🔍 【政策深度拆解】
📌 关税性质：25%从价税，针对重型机械整机
📌 豁免红线：非美产钢铝铜总重量<净重15%
📌 实质：穿透式原产地核查，封堵"近岸组装"规避路径

💡 【影响穿透】
大盘影响：国内整机出口面临25%关税重压，近岸组装模式失效
我司影响：墨西哥工厂若仅做总装，核心结构件仍从国内发货，重量占比将超15%红线，关税规避效果有限

⚠️ 【战略提示】
深度本地化制造（非简单总装）或成为唯一有效路径
```
