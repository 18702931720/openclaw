# 高空作业平台分析案例库

## 案例一：宏观政策深挖——美国232钢铝新政

### 资讯原文
"美国执行232钢铝新政，对重型机械整机征收25%关税，但非美产钢铝铜总重量低于净重15%可申请豁免。"

### 分析框架

**A. 政策深度拆解**

| 维度 | 内容 |
|------|------|
| 政策性质 | 232关税是美国出于国家安全的贸易保护壁垒 |
| 关税税率 | 25%从价税（针对整机） |
| 豁免条件 | 非美产钢、铝、铜总重量<净重15% |
| 核查方式 | 穿透式核查——从"整机原产地"深入到"底层原材料材质单" |
| 核心逻辑 | 封堵"国内生产核心结构件+周边国家简单组装"的传统规避路径 |

**B. 影响穿透**

| 层面 | 影响 |
|------|------|
| 大盘影响 | 高度依赖国内钢铁供应链的整机出口，大概率面临25%关税重压 |
| 我司影响 | 墨西哥工厂若仅做总装，核心结构件仍从国内海运发货，钢结构重量占比将远超15%红线，无法有效规避关税 |
| 战略提示 | 深度本地化制造（非简单总装）或成为唯一有效路径 |

## 案例二：行业并购拆解——JLG收购Power Tower

### 资讯原文
"全球高机巨头吉尔吉（JLG）已全资收购Power Tower。"

### 分析框架

**A. 标的分析与商业逻辑**

| 维度 | 内容 |
|------|------|
| 标的背景 | Power Tower是欧洲专注于室内低空作业平台（LLA）的头部厂商，主要产品为替代传统脚手架和梯子的无动力或微型推拉式升降机 |
| 市场定位 | 室内非机动升降设备，高毛利增量蓝海 |
| 并购意图 | JLG在传统室外重型高机市场饱和的背景下，利用资金优势快速切入室内设施维护赛道，完成全场景产品谱系的垄断 |

**B. 影响穿透**

| 层面 | 影响 |
|------|------|
| 大盘影响 | 巨头正加速向室内非机动升降设备这一细分市场倾斜资源 |
| 我司影响 | 我司正推进插装车、单手推动式检查设备等同类室内轻型产品，JLG收购后将利用全球顶级租赁网络进行排他性铺货，留给我司在欧美寻找中立代理商的渠道"真空期"正在急剧缩短 |
| 战略提示 | 该产品线的商业价值已被印证，需加速市场卡位 |

## 案例三：竞对商务动作——超长期分期政策

### 资讯原文
"徐工、中联等竞品在华中和区域下沉，针对优质客户推出30%首付比例及5–10年超长期分期政策。"

### 分析框架

**A. 商业手段分析**

| 维度 | 内容 |
|------|------|
| 动作内容 | 30%首付 + 5-10年超长期分期 |
| 设备生命周期 | 通常5-8年 |
| 动作实质 | 脱离设备销售范畴，转变为依靠集团雄厚资金池的"金融杠杆战" |
| 大盘影响 | 激进的账期政策能极大缓解租赁客户每月现金流压力，缺乏同等资金支撑的中小主机厂将面临优质大客户被强行绑定的风险 |

**B. 我司影响透视**

| 维度 | 内容 |
|------|------|
| 竞品手段 | 利用超长分期降低客户"即期资金压力"，意图通过金融工具长期锁定优质客群 |
| 我司模式 | 主推"2+2（2年质保+2年延保）"，解决客户中后期"高昂维保成本压力" |
| 竞争焦点 | 竞品的"前端金融减负" vs 我司的"后端服务兜底"，谁能更有效击中租赁客户全生命周期成本（TCO）痛点 |
| 战略提示 | 在资金账期上与巨头硬拼会严重拖累现金流与风控健康度，应坚持服务兜底差异化路线 |
