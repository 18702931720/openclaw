# 高空作业平台分析案例库

## 案例一：宏观政策深挖——美国232钢铝新政

### 资讯原文
"美国执行232钢铝新政，对重型机械整机征收25%关税，但非美产钢铝铜总重量低于净重15%可申请豁免。"

### ① 资讯内容分析
| 维度 | 内容 |
|------|------|
| 政策性质 | 232关税是美国出于国家安全的贸易保护壁垒 |
| 关税税率 | 25%从价税（针对整机） |
| 豁免条件 | 非美产钢、铝、铜总重量<净重15% |
| 核查方式 | 穿透式核查——从"整机原产地"深入到"底层原材料材质单" |
| 核心逻辑 | 封堵"国内生产核心结构件+周边国家简单组装"的传统规避路径 |

### ② 影响穿透
**大盘**：国内整机出口面临25%关税重压，近岸组装模式失效，高度依赖国内钢铁供应链的整机厂商承压

**星邦智能（我司）**：墨西哥工厂若仅做总装，核心结构件仍从国内海运发货，钢结构重量占比将远超15%红线，无法有效规避关税

### ③ 战略提示
- 深度本地化制造（非简单总装）或成为唯一有效路径
- 需重新评估墨西哥工厂的产能规划与供应链布局
- 时间窗口：政策已落地，需在未来3-6个月内给出应对方案

## 案例二：行业并购拆解——JLG收购Power Tower

### 资讯原文
"全球高机巨头吉尔吉（JLG）已全资收购Power Tower。"

### ① 资讯内容分析
| 维度 | 内容 |
|------|------|
| 标的背景 | Power Tower是欧洲专注于室内低空作业平台（LLA）的头部厂商，主要产品为替代传统脚手架和梯子的无动力或微型推拉式升降机 |
| 市场定位 | 室内非机动升降设备，高毛利增量蓝海 |
| 并购意图 | JLG在传统室外重型高机市场饱和的背景下，利用资金优势快速切入室内设施维护赛道，完成全场景产品谱系的垄断 |
| 整合逻辑 | 利用全球顶级租赁网络进行排他性铺货 |

### ② 影响穿透
**大盘**：巨头正加速向室内非机动升降设备这一细分市场倾斜资源，市场集中度将进一步提升

**星邦智能（我司）**：我司正推进插装车、单手推动式检查设备等同类室内轻型产品，JLG完成收购后，欧美渠道真空期正在急剧缩短，市场卡位窗口期收窄

### ③ 战略提示
- 该产品线的商业价值已被行业龙头验证
- 需加速欧美中立代理商的布局
- 室内轻型产品的渠道建设优先级应提升

## 案例三：竞对商务动作——超长期分期政策

### 资讯原文
"徐工、中联等竞品在华中和区域下沉，针对优质客户推出30%首付比例及5–10年超长期分期政策。"

### ① 资讯内容分析
| 维度 | 内容 |
|------|------|
| 动作内容 | 30%首付 + 5-10年超长期分期 |
| 设备生命周期 | 通常5-8年 |
| 动作实质 | 脱离设备销售范畴，转变为依靠集团雄厚资金池的"金融杠杆战" |
| 大盘影响 | 激进的账期政策能极大缓解租赁客户每月现金流压力，缺乏同等资金支撑的中小主机厂将面临优质大客户被强行绑定的风险 |

### ② 影响穿透
**大盘**：优质大客户被竞品长期锁定，中小主机厂客户流失加剧，行业集中度提升

**星邦智能（我司）**：在资金账期上与徐工/中联硬拼会严重拖累现金流与风控健康度；但我司主推的"2+2（2年质保+2年延保）"模式解决的痛点不同——竞品攻"前端金融减负"，我司守"后端服务兜底"

### ③ 战略提示
- 坚持服务兜底差异化路线，避免资金账期恶性竞争
- 将竞争焦点引导至"TCO全生命周期成本"而非"首付门槛"
- 深化"2+2"延保服务的市场认知，以"后端保障"对抗"前端减负"
