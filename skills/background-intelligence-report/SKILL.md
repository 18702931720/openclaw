# 背景情报调查报告技能

## 概述

本技能通过多轮搜索，自动收集目标公司的全方位情报，生成结构化调查报告。

## 搜索角度体系（6个维度）

| 维度 | 搜索关键词模式 | 重点提取 |
|------|--------------|---------|
| **基本信息** | `{公司名}` `股票代码` `成立时间` `法定代表人` `主营业务` | 公司全称、股票代码、成立时间、法人代表、注册地址、员工规模、主营业务描述 |
| **财务资金** | `{公司名}` `2025年报` `营收` `净利润` `现金流` `毛利率` | 营业总收入、净利润、经营活动现金流、海外收入及占比、分产品收入、主要财务指标同比变化 |
| **股权架构** | `{公司名}` `实际控制人` `控股股东` `子公司` `股权结构` | 实际控制人、控股股东、持股比例、主要子公司/关联公司、关联网络 |
| **司法风险** | `{公司名}` `诉讼` `处罚` `仲裁` `被执行人` `限制高消费` | 涉案金额、案件类型、审理状态、处罚原因、风险评级 |
| **核心人物** | `{公司名}` `董事长` `CEO` `总裁` `创始人` `高管` `薪酬` | 姓名、职务、背景简介、持股情况、风险标记 |
| **行业地位** | `{公司名}` `市场份额` `行业排名` `竞争对手` `行业地位` | 市场排名、竞争对手、战略优势/劣势、行业口碑 |

## 调查流程

### 第1步：基本信息收集

使用 SearXNG 执行初始搜索，收集公司基本情况。

### 第2步：财务深度搜索

搜索最近年度报告、季报数据，重点关注：
- 营收、净利润、现金流
- 海外收入占比
- 分产品/分地区收入构成

### 第3步：司法风险检索

搜索法律诉讼、行政处罚、仲裁等记录。

### 第4步：核心人物与股权结构

搜索管理层信息、股权结构、关联公司网络。

### 第5步：行业地位与舆情

搜索市场份额、行业排名、竞争对手对比、媒体报道。

### 第6步：汇总生成报告

整合所有信息，按标准格式输出调查报告。

## 配置要求

- **搜索工具**：SearXNG（隐私优先的元搜索引擎）
- **生成模型**：minimax/MiniMax-M2.7（或用户指定模型）
- **搜索轮次**：每维度至少 1-2 轮搜索，信息不足时追加搜索

## 使用示例

```
用户：帮我做一份宁德时代的背景情报调查报告
Agent：执行 background-intelligence-report 技能
       → 调用 SearXNG 搜索宁德时代多维度信息
       → 汇总生成结构化报告
```

## 注意事项

1. **信息交叉验证**：同一事实从多个来源交叉验证，提高置信度
2. **数据时效性**：财务数据优先使用最新年报/季报，新闻优先使用近期报道
3. **置信度标注**：对存疑信息标注来源和置信度级别
4. **平衡呈现**：正面和负面信息均需收集，避免偏见
5. **结论可操作**：最终结论应具有实际参考价值，而非仅是信息堆砌
