# 企业年度财报研究技能

## 一、技能定位

系统化研究企业年度财报的方法论，为投资、竞对分析、供应商评估等场景提供完整工作流。

**适用场景：**
- 投资尽职调查
- 竞争对手分析
- 供应商/客户背调
- 行业研究
- 信用评估

---

## 二、财报三张表核心框架

| 报表 | 核心问题 | 关键科目 |
|------|----------|----------|
| 资产负债表 | 家底有多厚 | 总资产、净资产、负债结构 |
| 利润表 | 赚钱能力有多强 | 营收、毛利率、净利润 |
| 现金流量表 | 钱是不是真进来 | 经营性现金流、投资性现金流 |

**附注是魔鬼**：应收账、存货、商誉、关联交易等异常往往藏在附注里。

## 三、研究步骤（五步法）

### Step 1：获取财报

| 来源 | 适用 |
|------|------|
| 巨潮资讯网 (cninfo.com.cn) | A股财报官方披露 |
| 东方财富 (eastmoney.com) | 财务数据汇总、PDF下载 |
| 港交所披露易 (hkexnews.hk) | 港股财报 |
| SEC EDGAR | 美股财报 |
| 雪球 | 投资者分析汇总 |

### Step 2：提取关键指标

**必读指标清单：**

```
盈利能力：
- 营收同比增速
- 毛利率 (%）
- 净利率 (%）
- ROE（净资产收益率）

成长性：
- 营收增速（近3年）
- 净利润增速（近3年）

质量：
- 经营性现金流 / 净利润（>1 为优）
- 应收账款增速 vs 营收增速
- 存货增速 vs 营收增速

安全性：
- 资产负债率
- 流动比率
- 速动比率

效率：
- 存货周转天数
- 应收账款周转天数
- 总资产周转率
```

### Step 3：识别危险信号（避雷优先）

| 危险信号 | 可能含义 |
|----------|----------|
| 净利润涨但经营性现金流为负 | 纸上富贵 |
| 应收账款增速 >> 营收增速 | 渠道压货或虚增收入 |
| 存货异常增加 | 产品滞销或成本虚减 |
| 毛利率持续下降 | 竞争加剧或议价能力弱 |
| 频繁更换会计事务所 | 可能有问题 |
| 大股东质押比例过高 | 资金链紧张 |
| 商誉占总资产 > 30% | 并购雷区 |

### Step 4：行业对比（找基准）

找到同行业2-3家头部公司横向对比：

| 对比维度 | 关注点 |
|----------|--------|
| 毛利率 | 谁赚钱更轻松 |
| 净利率 | 谁费用控制更好 |
| 周转天数 | 谁运营效率高 |
| 资产负债率 | 谁更稳健 |

### Step 5：历史趋势（3-5年）

单一年份可能是偶然，持续趋势才是真相：
- 营收是否稳定增长
- 毛利率是否稳定
- 现金流是否匹配利润
- 负债率是否可控

## 四、行业专用研究框架

### 高空作业平台行业（AWP）

**重点关注指标：**
- 营收结构：剪叉/臂车/桅柱比例
- 海外收入占比：判断国际化程度
- 产品毛利率：判断产品竞争力
- 应收账款周转：租赁商回款风险

**竞对参照：**
- 临工重机（LGMG）
- 徐工
- 中联重科
- 浙江鼎力（Dingli）
- JLG / Genie（国际）

### 其他行业

根据目标公司所处行业，调整关注重点（可参考 STEP 2 中的指标清单灵活选用）。

【关键指标】

## 七、注意事项

1. **先看附注再看报表**：异常往往藏在附注里
2. **跨期对比**：单期数据意义有限，3-5年趋势才可靠
3. **行业对标**：不同行业指标基准差异大，必须对比同行业
4. **识别口径**：注意非经常性损益、会计政策变更等干扰因素
5. **真实性验证**：多项指标异常时需警惕财报质量问题
