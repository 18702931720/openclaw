## 使用方法

### 基础发送

```python
python3 /home/admin/.openclaw/workspace/scripts/send-email.py \
  "recipient@sinoboom.com.cn" \
  "邮件主题" \
  "邮件正文内容"
```

## 安全注意

1. ⚠️ 发送前确认收件人邮箱地址
2. ⚠️ 敏感信息使用加密或内部渠道
3. ⚠️ 群发邮件注意隐私保护（使用 BCC）
4. ✅ 自动记录发送日志

## 错误处理

### 常见错误

| 错误 | 原因 | 解决方案 |
|------|------|----------|
| 550 User not found | 收件人不存在 | 确认邮箱地址 |
| 554 Message rejected | 内容被拒 | 检查敏感词 |
| Connection timeout | 网络问题 | 重试或检查网络 |
| Authentication failed | 认证失败 | 检查授权码 |
