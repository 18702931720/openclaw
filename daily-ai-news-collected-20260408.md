# 每日 AI 资讯收集 - 2026-04-08

## 国内资讯（6条）

### 1. Anthropic 发布 Claude 4.5
Anthropic 正式发布 Claude 4.5，复杂推理准确率提升 35%，支持 200万 token 长上下文，专为 企业级应用优化。新版本增强多模态理解，可处理文档、表格和图像混合内容。

### 2. 百度文心大模型 4.0 正式上线
百度发布文心大模型 4.0，中文语义理解准确率提升至 92%，代码生成支持超过 30 种编程语言，多轮对话记忆能力大幅增强。

### 3. OpenAI 发布 DALL-E 4
DALL-E 4 分辨率提升至 4096×4096，风格控制更精细，支持视频生成功能，安全性也同步改进。

### 4. 微软 Azure AI 大降价
GPT-4 API 价格下调 60%，企业采用成本大幅降低，进一步加速 AI 企业级落地。

### 5. 英伟达发布 Blackwell 架构
专为 AI 训练优化，性能提升 3 倍，为下一代大模型提供更强算力支撑。

### 6. 阿里通义千问 3.7 预告
即将支持 200万 token 超长上下文，在长文档处理方面挑战 Claude霸主地位。
